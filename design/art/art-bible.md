## Section 1: Visual Identity Statement

### The One-Line Rule

> **记忆是有质感的。越古老的，越不完整。**
> *Memory has texture. The further back, the more worn.*

每一个视觉决策——光照、色彩、材质、构图——都服从于这条规则：**盖亚所在的现在是最真实的，时间越往前，现实感越低，风化感越强**。这不是风格化，这是叙事。

### 三条支撑原则

**原则 1：时代即滤镜（Layer = Filter）**
四个时间层各有独立的视觉语言，彼此不兼容。进入任何一个时代，视觉系统必须立刻传达"这不是现在"。*设计测试：如果截图去掉 UI，玩家能在 3 秒内判断这是哪个时代，视觉系统成立。*

**原则 2：遗迹比任何角色都有故事（Ruins outlive their makers）**
环境的信息密度永远高于角色。墙上的刻文、倒塌的柱子、长满植物的水渠——这些是游戏真正的叙事载体。角色不应该喧宾夺主。*设计测试：如果一个空场景没有 NPC 时也能让玩家停下来看 30 秒，这个环境设计成立。*

**原则 3：猫族世界的尺度（Cat-scale world）**
所有建筑和道具都按照猫人的体型比例设计——门框低、台阶浅、座椅宽而矮。这不是装饰，是世界观的物理体现。*设计测试：人类站进这个场景会感觉微微不适（太低），但猫人会感觉刚好。*

## Section 2: Mood & Atmosphere

### 四时代情绪地图

---

**现在层（Present — 盖亚的时代）**
- **情绪目标**：温暖的好奇感。有过去的重量，但充满生机。
- **光照特征**：黄金时段光（清晨或傍晚），色温 5500–6500K，柔和定向阴影，Lumen 全局光照开启
- **大气描述词**：流动的 / 充盈的 / 略带尘埃的 / 草木的 / 有风的
- **色调基调**：暖橙+青绿双主色，高饱和度，高对比度中间调
- **节奏感**：沉稳、探索性——没有紧迫感，有重量

---

**旧世界末期（Old World — 父权崩塌前）**
- **情绪目标**：腐烂的华贵感。繁荣的外壳下暗藏腐坏。像一张旧照片——美丽，但知道它即将结束。
- **光照特征**：室内人工光（蜡烛/电光混合），色温 3200–4000K，过曝高光，阴影偏暖棕
- **大气描述词**：沉闷的 / 发黄的 / 过度精致的 / 闷热的 / 快要窒息的
- **色调基调**：低饱和度暖褐（Saturation ≈ 0.45），深色阴影，高光过曝，Film Grain 0.3
- **节奏感**：缓慢、压抑——每一个场景都感觉在憋着什么

---

**逃亡之路（Exodus — 废土时期）**
- **情绪目标**：极度的、无从逃脱的压迫感。不是悲伤，是恐惧和疲惫。这段历史是一道疤。
- **光照特征**：漫射日光（烟雾/灰烬散射），高对比度，强侧光，无暖色
- **大气描述词**：灰烬的 / 割裂的 / 坚硬的 / 疲惫的 / 锋利的
- **色调基调**：近黑白（Saturation ≈ 0.15），冷蓝灰，强 Vignette 0.7，Film Grain 0.5
- **节奏感**：紧张、间歇性爆发——静止与突变交替

---

**建国初代（Founding — 摩梭母系建立期）**
- **情绪目标**：粗糙的温柔感。一切都是第一次，都是用手做的，但充满了决心和爱。像一本手绘的书。
- **光照特征**：营火/油灯暖光，色温 2800–3500K，软阴影，光斑自然散射
- **大气描述词**：手工的 / 不完美的 / 温暖的 / 新生的 / 有烟火气的
- **色调基调**：中饱和度暖黄（Saturation ≈ 0.7），轻微纸质纹理感，Rounded Vignette 0.3
- **节奏感**：不稳定、但有希望——像第一批火种刚点燃

---

### 跨时代的通用大气规则

- 所有时代的**天空**都保留某种程度的美——即使在逃亡层，天空是灰白的，但有质感
- 每个时代的**植物**都存在（不同程度）——生命的韧性是这个世界的底层叙事
- **建筑废墟**是所有时代的共同语言，不同时代展示同一遗迹的不同阶段

## Section 3: Shape Language

### 角色剪影哲学

**盖亚（Maine Coon）**
- 大而蓬松的轮廓，耳簇明显，尾巴体积感强
- 首饰（金耳环、项链串珠）作为视觉识别符，在任何距离都可辨认
- 旅行装束的褶皱和宽松感与首饰的精致形成反差——"不修边幅但有来历"

**各族猫人的剪影规则**
- 每个品种的剪影在 64×64 像素缩略图下可辨认
- 暹罗猫（Naya）：修长、尖耳、尾巴纤细——速度和灵活感
- 无毛猫（Thal）：宽肩、圆头、存在感沉稳——智慧和重量
- 黑猫（Klou）：修长、几乎消失在阴影中——但眼睛是最亮的部分

**英雄形状 vs 背景形状**
- 英雄形状：流线曲线（角色、重要文物、遗迹核心构件）
- 背景形状：重复几何体（废墟碎石、地面纹理、植被）
- 规则：主角在场景中永远是"最弯曲"的形状

### 环境几何语言

**母系文明建筑（现在层 + 建国层）**
- 主形：圆形、椭圆、弧线——社区感、包容性、无等级的空间
- 门洞为圆拱，广场为圆形，座位排列呈向心圆
- 没有金字塔或方尖塔（父权纪念碑的象征）

**旧世界建筑（旧世界层）**
- 主形：垂直线条、锐角、精确对称——权威、控制、压制
- 对比：相同的建筑骨架，在现在层已经坍塌成曲线（时间消解了直线）

**废土（逃亡层）**
- 主形：破碎的直线、锈蚀曲面——人工物被自然力弯折
- 车辆、铁架、钢缆——工业形状在有机侵蚀下变形

### UI 形状语法

UI 不模仿世界建筑风格——采用极简的圆角矩形系统，配合手写质感字体。理由：叙事游戏的 UI 是"盖亚的日记本"，不是"猫人文明的界面"。UI 的形状语言是**纸张和笔迹**，不是石头和木头。

### 主色板

|--------|------------|----------|-----------------|
| **遗迹赭石（Ruin Ochre）** | 主环境色，石材基础 | `#C4873A` | 废土后重建的大地 |
| **母系青（Matriarchal Teal）** | 主交互提示色，重要文物 | `#4AADAA` | 母系文明的象征色，传承与水 |
| **记忆金（Memory Gold）** | 盖亚首饰，碎片触发高光 | `#E8B84B` | 值得被记住的东西 |
| **旧照棕（Old Photo Brown）** | 旧世界层基础色调 | `#7A5C3E` | 腐败与逝去 |
| **逃亡灰（Exodus Ash）** | 逃亡层基础色调 | `#8A8F96` | 压迫、灰烬、疲惫 |
| **建国暖（Founding Warm）** | 建国层基础色调 | `#D4934E` | 第一代人的手工温暖 |
| **日志米（Journal Cream）** | UI 背景色，日志系统 | `#F5EDD6` | 纸张、记录、文字 |

### 语义色彩规则

| 颜色 | 语义 | 用途 |
|------|------|------|
| **母系青** | 可交互 / 可触发 | 文物高光、交互提示、碎片入口 |
| **记忆金** | 重要叙事节点 | 关键文物、NPC 首饰、主线道具 |
| **白/近白** | 记忆本身 | 碎片过渡时的白色高光爆发 |
| **深红（Blood Red）** | 危险/创伤 | 逃亡层的暴力场景，极少用 |
| **纯黑** | 禁止使用 | 用非常深的遗迹赭石或逃亡灰代替 |

### 区域色温规则

| 区域 | 主色温 | 特征色 |
|------|--------|--------|
| 源丘城（Sourceholm） | 暖（5500K） | 赭石 + 母系青 |
| 镜湖领地（Mirror Coast） | 中性偏冷（6200K） | 湖蓝 + 建国暖 |
| 迷宫废墟（The Labyrinth） | 冷（7000K） | 石灰白 + 深紫 |
| 沙骨城（Dry Vein） | 极暖干燥（4000K） | 沙黄 + 深赭石 |
| 铁树神殿（Ironroot Shrine） | 暗（烛光，2800K） | 建国暖 + 极深绿 |

### UI 色板

UI 使用与世界色板独立的一套：
- 背景：日志米 `#F5EDD6`
- 文字：深棕 `#3D2B1F`
- 强调：记忆金 `#E8B84B`
- 交互提示：母系青 `#4AADAA`

### 无障碍设计

- **母系青（交互色）vs 逃亡灰（背景）**：明度差 > 4.5:1，符合 WCAG AA
- **红绿色盲风险**：逃亡层不使用红色作为唯一危险信号；配合形状和声音双重提示
- **所有 UI 文字**：背景对比度 ≥ 4.5:1

## Section 5: Character Design Direction

### 盖亚（主角，Maine Coon）

> ✅ **已有批准概念图**（2026-04-25）— 下方所有规格均以该概念图为准。
> 概念图文件：`design/art/reference/gaia_concept_sheet.png`（五视图 + 道具展示 + 面部特写）

---

**毛发与体型：**
- 底色：银灰虎斑（black mackerel tabby on silver）——面部、手臂、腹部有清晰的深色条纹
- 腹部/胸口：白色毛发，在皮夹克开口处可见
- 尾巴：长而蓬松，银灰渐深，体积感强——静止时垂至地面，是背影识别的核心元素
- 耳簇：Maine Coon 标志性长耳毛，直立，轮廓明显
- 眼睛：琥珀金（高饱和，≈ `#E8A020`），在所有光照条件下保持可辨认；默认眉形轻微内蹙（好奇感）

**服装（已定稿，与概念图一致）：**

| 部位 | 描述 | 材质/颜色 |
|------|------|---------|
| 上衣 | 短款包裹式皮夹克，前开不扣，多层皮带扣装饰 | 深棕 / 巧克力色做旧皮革 |
| 胸口徽章 | 黄铜锻造纹章（类十字花形），缝于夹克左胸 | 记忆金 `#E8B84B` |
| 裤子 | 宽松工装棉裤，小腿处收紧 | 卡其 / 橄榄绿 |
| 小腿裹布 | 厚布条缠绕至踝，制造层次感 | 白/米色粗布 |
| 靴子 | 高筒系带皮靴（膝下），多条皮带扣横跨 | 深棕做旧皮革，黄铜扣 |
| 腰带系统 | 宽主腰带 + 多个小挂包 + 皮条下垂 | 棕色皮革，铆钉装饰 |
| 装饰腰布 | 从腰右侧垂挂的红/酒红色刺绣布面 | 红褐色织物，几何纹样 |
| 手套 | 露指皮手套 | 深棕做旧皮革 |

**首饰（视觉识别符，任何距离可辨认）：**
- 耳朵：双耳多枚金色圆环/月牙耳环，右耳有红色珠饰和青绿宝石点缀
- 项链：多层——金链 + 记忆金 `#E8B84B` 圆牌吊坠
- 腰带挂件：小铃铛、珠串、零散小挂饰

**随身道具（固定出现在场景中的考古学家套装）：**

| 道具 | 外观 | 叙事意义 |
|------|------|---------|
| 皮制单肩包 | 翻盖式邮差包，黄铜十字花锁扣，深棕做旧皮革 | 日志和碎片素材的实物载体 |
| 卷起的地图 | 旧羊皮纸材质，红色路线标注，边角磨损 | 玩家视觉上感知"已探索区域" |
| 黄铜罗盘 | 圆形，八角刻度，表面有划痕 | 母系青 `#4AADAA` 指针 |
| 小望远镜 | 单筒黄铜伸缩望远镜，约 15cm 展开 | 探索感的道具强化 |

**表情原则：**
- 默认表情：**好奇的轻蹙眉** — 不是微笑，不是沉默，是永远在想什么
- 耳朵是情绪主信号——高兴时前倾，警惕时直立，难过时向后贴
- 概念图中呈现的面部神情（琥珀眼直视、眉间微蹙）即为 LOD0 标准表情

**LOD 规则（Godot 4 实现）：**
- LOD0（近景 < 3m）：全细节毛发 mesh（静态，分层卡片毛），完整面部骨骼动画
- LOD1（3–10m）：简化毛发贴图，面部动画保留耳朵和眼睛
- LOD2（> 10m）：角色剪影 mesh，保留尾巴和首饰识别符
- *注：Godot 4 中通过 `VisualInstance3D.lod_bias` + 手动 mesh 切换实现*

### NPC 角色设计通则

**品种即角色类型：**

| NPC | 品种 | 视觉重点 | 颜色主色调 |
|-----|------|----------|-----------|
| 纳雅 | 暹罗猫 | 极细腰身，长颈，颜色渐变脸 | 象牙白 + 深棕 |
| 塔尔 | 无毛猫 | 宽肩，皮肤纹理，不戴首饰 | 淡粉肉色 + 深蓝布料 |
| 克娄 | 黑猫 | 全身最深色，眼睛是场景内最亮的单点 | 纯黑（高光处带紫色反光） |
| 玛里斯长老 | 波斯猫 | 蓬松压住眼睛，给人遮蔽感 | 银白 + 正式深紫 |

**区分法则：**
- 主线 NPC 戴首饰或有特征纹样
- 路人 NPC 服装色调在当前区域色板范围内，不跳脱
- 敌对/威胁性角色（逃亡层）：服装以工业材料为主，无曲线

### 碎片角色（Fragment Characters）

每个时代的碎片角色是独立的视觉设计，但遵循以下共同规则：
- 手部（交互的主要画面）必须清晰可读
- 服装颜色服从当前时代色板
- 玩家不会看到碎片角色的脸（第一人称视角或后脑勺视角）

## Section 6: Environment Design Language

### 建筑风格与文化关联

**母系文明遗迹（现在层主要场景）**

*参照原型：加泰土丘 / 泸沽湖 / 米诺斯宫殿*

- **平面逻辑**：无轴线，无对称，走廊随机偏转——迷路感是刻意的，象征历史的非线性
- **材质层次**：底层为粗石（古），中层为抹灰（中古），表层为现代修缮（新），三层叠加可见
- **植物侵入**：所有遗迹均有植物生长入缝隙，量级按废弃时间递增——铁树神殿最多，沙骨城最少
- **水的痕迹**：苔藓、水渍、古水渠——水是母系文明的核心资源，处处留痕

**旧世界残留物（旧世界层碎片场景）**

*参照原型：19–20世纪工业/殖民建筑废墟*

- 垂直线条主导，金属外框配石材内衬
- 家具体型超出猫人尺度（旧世界的权力者忘记了服务自己的族群）
- 所有光源（灯、窗）都略微过度——追求明亮本身，而非温暖

**废土（逃亡层碎片场景）**

*参照原型：废弃工厂、高速公路废墟、集装箱改建*

- 没有遗迹，只有改造：一切都是"把别的东西拆了用"
- 临时感——没有一件东西是为了长久而建的
- 细节密度高（杂乱的道具）但色调极度统一（灰蓝）

**建国初代（建国层碎片场景）**

*参照原型：游牧定居初期，土坯+木构*

- 所有建筑都是人手能完成的尺度——没有起重机能搬动的石块
- 手痕可见：墙面的指印、不平整的木工接缝、用绳子捆扎的梁柱
- 光源全为有机火光（营火、油灯、火把）

### 纹理哲学

**现在层**：PBR 工作流，Substrate 材质，高粗糙度（matte 感），中等金属感。不追求反射光洁感，追求触摸感。

**旧世界层**：故意略微降低 PBR 精度——过曝高光、略微暗的阴影，营造"记忆中的印象"而非"真实的物理"。

**逃亡层**：锈蚀、破裂、表面脱落——纹理的主要信息是损坏，而非原始材质。

**建国层**：手绘纹理贴图风格（不完全写实，有笔触感），低法线细节，材质偏向：土、布、木。

### 道具密度规则

| 区域类型 | 密度 | 原则 |
|----------|------|------|
| 公共广场/入口 | 稀疏 | 让建筑说话，不用道具填充 |
| 居住/私人空间 | 中等 | 每件道具都有叙事意义 |
| 仪式/神圣空间 | 极稀疏 | 空白即神圣 |
| 废土/逃亡场景 | 密集 | 杂乱是叙事语言，但色调统一 |

### 环境叙事准则

每个场景至少包含一个"只有认真看才会发现"的细节：
- 一件私人物品（暗示这里曾有具体的人）
- 一个时代痕迹的冲突（旧世界的东西在新文明的空间里）
- 一个悬而未决的符号（玩家在看完某个碎片后才会明白）

## Section 7: UI / HUD Visual Direction

### 设计哲学：盖亚的日记本

所有 UI 被定义为"盖亚随身携带的日记本"——不是猫族文明的技术界面，而是一个旅行者的手写记录。这意味着：
- 字体：带轻微手写感的衬线字体（中文：方正仿宋或类似；英文：Playfair Display 类）
- 颜色：日志米底色 `#F5EDD6` + 深棕墨水 `#3D2B1F`
- 边框：手绘风格的不规则线条，轻微倾斜
- 动画：翻页感（非弹窗，非滑入——像翻书页）

### HUD 元素

| 元素 | 显示条件 | 视觉风格 | 位置 |
|------|----------|----------|------|
| 交互提示 | 焦点目标存在时 | 母系青标签 + 手写字体，细描边 | 屏幕中下方 |
| 检视文本 | 长按检视触发 | 日志米底条，深棕文字，淡入淡出 | 屏幕下方 1/3 |
| 存档图标 | 存档进行时 | 极小羽毛笔图标，角落 | 右下角 |
| 日志解锁提示 | 新条目解锁时 | 轻微脉冲的日记本图标，2秒后消失 | 右下角 |

**HUD 在碎片进行中完全隐藏。**

### 主菜单 / 暂停菜单

- 背景：模糊的当前/最近游戏画面（不是黑屏）
- 菜单为竖向日记本样式，每个选项是一行手写条目
- 动画：像翻到新一页，不用弹窗或滑板

### 日志/编年史 UI

- 双页摊开设计（左页：时间线；右页：当前条目内容）
- 时间线用竖轴表示，四条并行（四个时代各一条），碎片完成后在对应时代轴上添加一个节点
- 节点样式：手绘圆圈，完成的实心，未发现的虚线（仅在发现后才出现）
- 文字风格：盖亚的第一人称内心独白，不是百科全书式描述

### 无障碍要求

- 所有 UI 文字最小 16px（1080p 基准），可在设置中放大至 24px
- 字体颜色与背景对比度 ≥ 4.5:1
- 交互提示不仅用颜色区分，还用图标形状区分（可用/不可用有不同图标轮廓）

## Section 8: Asset Standards

### 文件命名规范

```
[Type]_[Area/Character]_[Description]_[Variant].[ext]

类型前缀：
  SM_   静态网格体（Static Mesh）
  SK_   骨骼网格体（Skeletal Mesh）
  T_    纹理（Texture）
  M_    材质（Material）
  MI_   材质实例（Material Instance）
  BP_   蓝图（Blueprint）
  ABP_  动画蓝图（Animation Blueprint）
  LVL_  关卡（Level）
  DA_   数据资产（Data Asset）
  SFX_  音效（Sound）
  MUS_  音乐（Music）

示例：
  SM_Sourceholm_ArchWall_Cracked
  T_Gaia_Body_Albedo_01
  MI_Fragment_OldWorld_Stone
```

### 纹理分辨率标准

| 资产类型 | 高配（PC Ultra） | 中配（PC Medium/Console） | 纹理格式 |
|----------|-----------------|--------------------------|----------|
| 主角角色 | 4096×4096 | 2048×2048 | BC7 (Albedo/Roughness), BC5 (Normal) |
| NPC（主要） | 2048×2048 | 1024×1024 | 同上 |
| NPC（路人） | 1024×1024 | 512×512 | BC1 |
| 建筑/遗迹（Nanite） | 4096×4096 | 2048×2048 | BC7 |
| 道具（英雄道具） | 2048×2048 | 1024×1024 | BC7 |
| 道具（背景道具） | 1024×512 | 512×256 | BC1 |
| UI 元素 | 1024×1024 | 512×512 | BC3 (需 Alpha) |
| 地面/地形 | 2048×2048 (tiling) | 1024×1024 | BC7 |

**Nanite 策略**：所有静态遗迹建筑启用 Nanite，角色和互动道具不使用 Nanite（需要精确碰撞）。

### 多边形预算

| 资产类型 | 目标面数（LOD0） | 备注 |
|----------|----------------|------|
| 盖亚（玩家角色） | 60,000–80,000 tri | Groom 毛发另计 |
| 主要 NPC | 30,000–50,000 tri | |
| 路人 NPC | 10,000–15,000 tri | |
| 英雄道具（文物） | 5,000–15,000 tri | |
| 背景道具 | 500–3,000 tri | |
| 建筑构件（Nanite） | 无上限（Nanite 处理） | |

### 材质规范

- 所有新材质使用 **Substrate Material System**（UE5.7 生产可用）
- 不使用 Legacy Unlit 材质（除 UI 和粒子）
- 每个材质实例最多 4 个纹理采样槽（性能限制）
- Time Layer 材质变体通过 Material Parameter Collection（MPC）统一控制，不创建独立材质

### LOD 规则

- 所有角色必须有 LOD0–LOD2（详见 Section 5）
- 建筑构件使用 Nanite 代替手动 LOD
- 道具 LOD：英雄道具 LOD0–LOD1，背景道具仅 LOD0（通过 Cull Distance 管理）

### 导出设置

- 网格体：FBX，Y-up，单位：厘米
- 纹理：PNG（导入前），引擎内转换为对应 BC 格式
- 动画：FBX，嵌入骨骼，30fps 导出（引擎内支持不同帧率重定向）

## Section 9: Reference Direction

### 参考来源与取用规则

---

**参考 1：《Journey》（thatgamecompany，2012）**
- **取用**：角色在宏大遗迹中的渺小感；清晰的剪影语言（沙漠背景下角色永远可识别）；沉默叙事——环境比文字更有力
- **避免**：极简到无道具的空旷感（盖亚的世界有历史细节）；无对话的纯意象叙事（游戏需要对话和选择）

---

**参考 2：《Roma》（Alfonso Cuarón，2018）**
- **取用**：旧世界层的视觉质感——黑白/低饱和下的生活细节；历史感的摄影构图（前景物件框住背景）；普通人视角下的大历史事件
- **避免**：完全去色（我们的旧世界层保留暖褐色调，不是黑白）；镜头感（游戏是第三人称互动，不是电影长镜头）

---

**参考 3：《Midsommar》（Ari Aster，2019）**
- **取用**：建国层的视觉语言——明亮的日光下反常的仪式感；纯手工制品的质感（花环、刺绣、陶器）；色彩饱满但不刺眼的"原始暖"
- **避免**：恐怖氛围（我们的建国层是温柔而非恐怖的）；过度的民俗堆砌（保持克制）

---

**参考 4：《奥本海默》序列 / 《银翼杀手 2049》色彩设计**
- **取用**：逃亡层的色彩——橙灰、冷蓝、高对比，颜色本身传达情绪而非物理真实；孤立人物在宏大破败环境中的对比感
- **避免**：未来感科技感（废土是没有科技的，只有锈铁）；人物的英雄主义构图（逃亡层的人不是英雄，是幸存者）

---

**参考 5：宫崎吾朗《来自红花坂》 + 高坂希太郎环境设计**
- **取用**：现在层的情绪基调——废旧之物中的温暖人情；植物与人造物的共存；生活感的环境细节（晾晒的衣物、种在罐头里的草）
- **避免**：日本现代建筑风格（猫族建筑是更接近非洲/中亚古代民居的弧形石构）；少年感主角（盖亚更世故，不是青春热血）
